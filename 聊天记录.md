**症状：** 你的日志里显示 `⏳ Retrying in 2.0s (attempt 1/3)...` 和 `❌ API failed after 3 retries — Connection error`，就是代理失效的表现。

**修复：**
1. 把 `~/.config/com.vortex.helper/config.yaml` 中的 `mode: direct` 改回 `mode: rule`
2. 重启Vortex进程（杀掉后自动重启，PID从8060变到32140）
3. 验证代理连通：`curl -x http://127.0.0.1:7897 https://api.openai.com/v1/models` 返回401（可达）

**这是之前记录过的经典故障：** Vortex的`mode: direct`会静默禁用所有代理路由，必须改回`mode: rule`。
