# BLIIoT S-Series / S150 蜂窝M2M RTU产品线

---

## S475/S475E — 高端蜂窝IoT RTU 🌟🌟

### S475（4G LTE版）
- **网络**: GSM/3G/4G LTE内置蜂窝模块，双SIM卡
- **处理器**: ARM Cortex-M4 32-bit 168MHz + RTOS
- **DI**: 8×DI（干/湿接点），DIN0高速计数器1MHz，DIN1-3低速计数器1KHz，DIN1可布防/撤防
- **AI**: 6×AI，**24位分辨率**，0-5V / 0-20mA / 4-20mA（可切换PT100温度检测）
- **⚠️ AI量程限制**: **不支持0-10V**！只有0-5V和0/4-20mA。客户需要0-10V时需变送器转换
- **DO**: 4×继电器输出（5A/30VDC, 5A/250VAC），支持脉冲输出
- **温湿度**: 1×外接AM2301传感器（-40~80°C, 0~99%RH）
- **RS485**: **2×RS485**（比S275多1个），支持Modbus RTU主站/从站+透传
- **以太网**: 1×RJ45（支持TCP Server，最多5个客户端连接）
- **协议**: SMS, GPRS/UDP, TCP, **MQTT**, Modbus RTU, Modbus TCP
- **存储**: 内置32G SD卡，保存10万条历史数据和事件
- **备用电池**: 3.7V 850mAh（可选）
- **电源**: 9~36VDC + 1路对外供电输出
- **映射寄存器**: 64布尔 / 64(16位) / 128(32位) / 64(64位)
- **尺寸**: 70×87×52mm, 350g
- **文档**: `产品规格书/英文资料/S47x/产品说明书/BLIIoT S475 S475E_usermanual_v1.6.4.pdf`

### S475E（以太网版，无蜂窝）
- 与S475相同I/O规格，但**没有蜂窝模块**（不能短信/拨号报警）
- 适合只需要以太网连接的场景

### 客户选型关键对比

| 特性 | S275 | S475 | S475E |
|------|------|------|-------|
| 蜂窝网络 | GSM/3G/4G | GSM/3G/4G | ❌ 无 |
| AI分辨率 | 12位 | **24位** | 24位 |
| AI量程 | 0-5V/0-20mA/4-20mA | 0-5V/0-20mA/4-20mA | 同S475 |
| **0-10V支持** | ❌ | ❌ | ❌ |
| RS485 | 1× | **2×** | 2× |
| 计数器 | 1×(DIN0) | **4×**(DIN0-3) | 4× |
| MQTT | ✅ | ✅ | ✅ |
| SD卡 | 8G | **32G** | 32G |
| 映射寄存器 | 最多320 | 最多320 | 最多320 |

### ⚠️ 选型陷阱：0-10V需求
S系列全部（S270-S475）的AI只支持 **0-5V 和 0/4-20mA**，**不支持0-10V**。
客户要求0-10V输入时，解决方案：
1. S475 + 0-10V→4-20mA变送器（每通道约$5-15）
2. MxxxT模块（支持0-10V）+ BL118网关（提供4G+MQTT+RS485）
3. MxxxE（4G版MxxxT，待确认上市时间）

## S275 — 4G IoT M2M RTU (ZODSAT项目使用) 🌟
- **网络**: 4G/3G/GSM（全球频段）
- **DI/DO**: 2×DI + 2×DO（继电器）
- **AI**: 2×AI (4-20mA/0-10V) ← PT100温度传感器用这个
- **RS485**: 1×RS485 (Modbus RTU主站/从站)
- **温湿度**: 内置温湿度传感器
- **报警**: 短信/拨号/APP推送
- **配置**: USB+PC软件 / SMS命令
- **日志**: 历史记录, GPRS远程下载
- **应用**: ✓变压器防盗监控, 泵站, 油箱液位, 环境监测
- **文档**: `产品规格书/英文资料/S27x系列/S275/`
- **注意**: 这是ZODSAT Zim变压器防盗项目使用的设备

## S271 — 3G GSM RTU
- 3G/GSM, 类似S275但仅3G
- 2×DI + 2×DO + 2×AI
- 应用: 基本远程监控

## S150 — 经济型GSM SMS控制器
- 2G/GSM网络
- 短信控制：布防/撤防/继电器开闭/查询状态
- 1×DI + 1×继电器输出
- 通过短信命令操作
- 应用: 安防报警, 自动监控系统, 水/油位, 泵站, 自动售货机
