# BLIIoT S27x/S150 系列蜂窝M2M RTU

## S275 — 4G IoT M2M RTU (ZODSAT项目使用) 🌟
- **网络**: 4G/3G/GSM（全球频段）
- **DI/DO**: 2×DI + 2×DO（继电器）
- **AI**: 2×AI (4-20mA/0-10V) ← PT100温度传感器用这个
- **RS485**: 1×RS485 (Modbus RTU主站/从站)
- **温湿度**: 内置温湿度传感器
- **报警**: 短信/拨号/APP推送
- **配置**: USB+PC软件 / SMS命令
- **日志**: 历史记录, GPRS远程下载
- **应用**: ✓变压器防盗监控, 泵站, 油箱液位, 环境监测
- **文档**: `产品规格书/英文资料/S27x系列/S275/`
- **注意**: 这是ZODSAT Zim变压器防盗项目使用的设备

## S271 — 3G GSM RTU
- 3G/GSM, 类似S275但仅3G
- 2×DI + 2×DO + 2×AI
- 应用: 基本远程监控

## S150 — 经济型GSM SMS控制器
- 2G/GSM网络
- 短信控制：布防/撤防/继电器开闭/查询状态
- 1×DI + 1×继电器输出
- 通过短信命令操作
- 应用: 安防报警, 自动监控系统, 水/油位, 泵站, 自动售货机
