# BLIIoT IPM100 IP67 远程IO模块

## 核心规格
- **防护**: IP67防水防震
- **接口**: RS485, Modbus RTU协议
- **DI**: 8路干接点输入（兼容干/湿接点）
- **型号**: IPM100(预铸线缆+灌胶)/IPM100T(外接电缆+免工具端子)
- **材质**: 耐酸碱, 工业级
- **安装**: 适合严苛户外/潮湿环境
