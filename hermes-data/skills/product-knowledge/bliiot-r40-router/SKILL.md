# BLIIoT R40 系列工业4G蜂窝路由器

## 型号
- **R40**: 基础款 4G工业路由器
- **R40A**: 增强型号A
- **R40B**: 增强型号B

## 核心规格
- **4G**: 内置工业级4G通信模块
- **以太网**: 3×LAN + 1×WAN (WAN可配置为LAN), 10/100Mbps
- **PoE PSE**: 可选 IEEE 802.3af/at 标准, 可给网络设备供电
- **WiFi**: STA+AP双模式, 2.4G+5G
- **双SIM卡**: 支持自动故障切换(有线/无线双链路)
- **VPN**: L2TP, IPSEC, OPENVPN
- **保护**: 独立硬件看门狗, EMC/EMI高低温测试
- **应用**: 无人值守机房, 智慧城市, 隧道/桥梁监控, 气象站, 森林防火, 水利
