# 🧸 IOy 系列 — 可编程边缘IO模块 （EdgeIO）

## 🎯 一句话核心
> **IOy = 传感器电线插进去 → 网络数据吐出来**

把物理世界的信号（开关按没按、温度多少度），变成网络世界的数据（Modbus TCP / OPC UA / MQTT...），而且内置逻辑判断，可以替代小型PLC。

---

## 📦 硬件规格（6款完全一样）

| 项目 | 参数 |
|------|------|
| 电源 | 24VDC，功耗仅55mA (~1.3W) |
| 网口 | 2×RJ45 10/100M，**支持级联**（菊花链） |
| 串口 | 1×隔离RS485（Modbus RTU主站模式，读智能电表等） |
| IO扩展槽 | 1~3个，插Y板自由组合 |
| 保护 | ESD ±6kV接触/±8kV空气，EFT 1kV/5kHz |
| 温宽 | -45°C ~ 80°C |
| 看门狗 | 独立硬件看门狗 + EMC/EMI + 电气隔离 |
| 安装 | DIN35导轨 |
| 外壳 | 塑胶壳 |

## 🔄 工作流程

```
[传感器/设备] → ①Y板采集信号 → ②内置逻辑判断 → ③协议转换 → [上位系统]
                                                ↓
                                           可选BLRAT远程维护
```

# 🎨 六兄弟深度解析

## 🟢 BL190 — Modbus TCP 版「万能通用型」

**语言**: Modbus TCP — 工业界最古老最通用的协议，像工业英语
**对接**: PLC / SCADA / HMI
**客户**: 工厂设备工程师、自动化集成商
**场景**: 工厂产线IO扩展，接到西门子PLC或上位机SCADA
**卖点**: 兼容性最强，配置最简单，价格通常最低

> Modbus TCP 就像可口可乐 🥤 —— 不是最好喝的，但全世界哪都有

## 🔵 BL191 — OPC UA 版「安全学霸」

**语言**: OPC UA (Unified Architecture) — 工业4.0官方语言
**特点**: ✅ X.509证书 ✅ AES-256加密 ✅ 符合IEC 62443工控安全
**对接**: MES / ERP / SCADA / 云平台（IT层系统）
**客户**: 数字化转型大厂、对安全有要求的客户（汽车/制药/食品）
**场景**: OT到IT的桥梁，打通产线到管理层
**卖点**: 自带加密+数据模型+安全合规

> OPC UA 就像顺丰快递 📦🔒 —— 不仅送货，还带加密包装、实时追踪

## 🟣 BL192 — MQTT 版「云宝宝」

**语言**: MQTT — 物联网界的微信，发布/订阅模式，主动推送
**直接支持**: 阿里云IoT、华为云IoT、AWS IoT、ThingsBoard
**对接**: 各类IoT云平台
**客户**: 远程监控、上云需求的客户、储能监控、环保监测
**场景**: 光伏电站远程监控，设备数据直接上云，手机APP随时看
**卖点**: 穿越防火墙、省带宽、断网自动重连

> MQTT 就像发朋友圈 📱✨ —— 一次发布，所有订阅者都能看到

## 🟠 BL193 — SNMP 版「IT小哥」

**语言**: SNMP — IT网管界的世界语，把工业IO变成MIB库
**对接**: Zabbix / SolarWinds / PRTG 等企业IT网管系统
**客户**: 企业IT部门（非工厂自动化部门！）、数据中心动环监控
**场景**: 数据中心机房温湿度/烟雾/门禁 → SNMP → Zabbix统一监控大屏
**卖点**: 给工业设备装「IT翻译器」，让网管系统能看懂

> SNMP 就像IT小哥的万能遥控器 🎮 —— 以前只能管电脑，现在能管传感器和门禁了

## 🔴 BE190 — IEC 104 版「电网老司机」

**语言**: IEC 104 — 电力行业标准协议，国家电网/南方电网都在用
**对接**: 变电站监控系统 / 电网调度中心
**客户**: 电力系统集成商、变电站/配电网项目、光伏/风电/储能
**应用场景**: ①智能变电站 ②配网自动化 ③新能源电站 ④电网调度 ⑤储能系统 ⑥充电桩
**注意**: BE190是IEC 104 Edge IO，硬件比BE102P强，支持CP56Time2a时间戳的可能性更大

> IEC 104 就像电网专属的身份证 🔌 —— 不认这个，变电站的门都进不去

## 🟡 BA190 — BACnet/IP 版「楼宇管家」

**语言**: BACnet/IP — 楼宇自控界国际语言，空调/灯光/门禁/电梯都懂
**对接**: 楼宇BMS（Building Management System）
**客户**: 楼宇自控集成商、商业大厦/智慧园区/医院/数据中心
**场景**: 办公楼灯光/空调/新风/门禁 → BACnet/IP → BMS统一管理
**卖点**: 支持级联（设备手拉手），楼宇项目必备

> BACnet/IP 就像大楼的管家 🎩 —— 不管你是空调工还是门禁工，都得跟管家汇报

## 🧩 Y板系统 — 核心卖点

最多插3块Y板，混搭随意。像吃火锅自己调蘸料 🍲

| Y板 | 功能 | 通道 | 说明 |
|-----|------|------|------|
| Y01/Y02 | DI数字量输入 | 8 | 干接点/湿接点(NPN/PNP) |
| Y11/Y12/Y13 | DI数字量输入 | 8 | 更多类型 |
| Y21/Y22 | DO数字量输出 | 8 | 开关控制 |
| Y24 | 继电器输出 | — | 大功率开关 |
| Y31/Y33 | AI电压输入 | 4 | 模拟量采集 |
| Y34/Y36 | AI电压输入 | 4 | 更多量程 |
| Y41 | AO电流输出 | 4 | 控制阀门等 |
| Y43/Y46 | AO电压输出 | — | 电压输出 |
| Y51-Y54 | RTD热电阻 | 2 | PT100/PT1000 |
| Y58 | TC热电偶 | — | J/K/T/E/R/S/B/N型 |
| Y61 | 串口扩展 | 4 | RS485/RS232 |
| Y95 | 脉冲输出/计数 | — | 频率/脉冲 |

### 组合示例
- 3×Y11 = **24DI** 纯输入模块
- Y11(8DI) + Y31(4AI) + Y52(2RTD) = **8DI+4AI+2PT** 混合模块
- Y61(4×RS485) + Y24(继电器) + Y11(8DI) = 多串口+开关控制

### 🎯 "All-in-One" PLC替代组合（客户常见需求）

当客户说"我要一个模块同时搞定DI+AI+TC+脉冲"时，用 **BL118B + Y板组合** 方案：

| 客户需求 | 推荐Y板 | 说明 |
|---------|---------|------|
| 2-4路DI | Y01/Y02 (4DI+4DO) | NPN/PNP可选 |
| 1-2路AI 4-20mA | Y31 (4AI, 0/4~20mA) | 单端输入 |
| 1-2路AI 0-10V | Y33 (4AI, 0-5V/0-10V) | 单端输入 |
| 1-4路热电偶 | Y58 (4TC, B/E/J/K/N/R/S/T) | 全类型支持 |
| 编码器/脉冲输入 | Y95 (4PWM+4脉冲计数) | ⚠️ 脉冲计数✅ 但**不是正交编码器(A/B相)** |

**完整推荐配置：BL118B-SOM334-X4-Y02-Y33-Y58-Y95 + 4G模块**

| 部件 | 功能 | 价格 |
|:----|:-----|:----:|
| BL118B主机 | 双核CA7@1.2GHz, Node-RED+BLIoTLink | $58 |
| SOM334 | 512MB RAM, 8GB eMMC | $30 |
| X4 | 2×RS485 + 2×CAN | $8 |
| Y02 | 4DI+4DO PNP | $10 |
| Y33 | 4AI 0-5V/0-10V | $18 |
| Y58 | 4TC热电偶 | $17 |
| Y95 | 4脉冲输出+4脉冲计数 | $23 |
| 4G模块 | 按客户地区选频段 | $9~$33 |
| **合计** | | **~$164 + 4G** |

**⚠️ 注意：** Y95是**脉冲计数**（高速1路+低速3路），不是**正交编码器（A/B相）**。如果客户要的是真正的编码器（测速/定位），Y95只能计脉冲数，不能判断方向。需要跟客户确认"encoder input"的具体含义。

**⚠️ 如果客户不需要4G远程上传：** 可以用IOy系列（BL190/BL191/BL192等）代替BL118B，价格更低但只有以太网没有4G。
