# BLIIoT BL120/BL121 系列工业自动化网关

## BL120 系列 — Modbus 工业自动化网关
- **型号**: BL120, BL120AC, BL120BN, BL120DT, BL120ML, BL120PM 及各PoE版本(P后缀)
- **处理器**: 300MHz ARM MCU
- **RAM**: 128MB, **Flash**: 64MB
- **以太网**: 2×RJ45 10/100Mbps
- **串口**: 2CH RS485/RS232
- **4G**: 可选(L-E/L-CE/L-A/L-AU/L-AF/CAT-1)
- **GPS**: 可选
- **SIM**: 1×NANO
- **供电**: DC 9~36V
- **尺寸**: 30×83×110mm
- **保护**: ESD ±6kV, EFT 1kV

## BL121 系列 — OPC UA 工业自动化网关
- **型号**: BL121, BL121AC, BL121BN, BL121DT, BL121ML, BL121P, BL121PO, BL121POP 及PoE版
- **处理器**: 300MHz ARM MCU
- **RAM**: 128MB, **Flash**: 64MB
- **以太网**: 2×RJ45 10/100Mbps
- **串口**: 2CH RS485/RS232
- **WiFi**: 可选 802.11a/b/g/n, 2.4G+5G
- **协议**: Modbus RTU/TCP↔OPC UA + MQTT
- **其他**: 同上BL120

## BL122 — Modbus↔PROFINET 网关
- 协议转换 Modbus RTU/TCP ↔ PROFINET

## BL123 — Modbus↔EtherCAT 网关
- 协议转换 Modbus RTU/TCP ↔ EtherCAT

## BL124 — Modbus↔EthernetIP 网关
- 协议转换 Modbus RTU/TCP ↔ Ethernet/IP

## BE120 — Modbus↔IEC 61850 网关
- **处理器**: 双核Cortex-A7 @1.2GHz
- **RAM/存储**: 可选1GB RAM + 8GB存储
- **串口**: 4~8路隔离RS485/232
- **协议**: Modbus RTU/TCP ↔ IEC 61850 (SCL文件配置)
- **温宽**: -45°C ~ 80°C
- **特点**: 双网口+可选WiFi/4G/5G, 钛合金外壳
- **应用**: 智能变电站, 新能源发电数据采集
