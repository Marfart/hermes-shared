## 🎯 雅思听力备考策略

1. **精听**：用EchoScribe转录→逐句跟读→对比原文
2. **泛听**：每天听1小时英语播客（通勤/休息时）
3. **填空练习**：剑桥真题Section 1-4填空题
4. **拼写训练**：听力答案拼写必须100%正确

## 🎯 雅思阅读备考策略

1. **Skimming**：快速浏览标题+首尾句，抓文章结构
2. **Scanning**：定位关键词找具体信息
3. **T/F/NG题**：区分False（相反）和Not Given（未提及）
4. **时间管理**：3篇文章60分钟=每篇20分钟
