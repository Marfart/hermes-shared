# 多角色 Agent 编排器

## 模式
CrewAI/LangGraph风格的角色分离+状态机

## 角色
1. **客户开发** — LinkedIn搜索 + 目录推广
2. **客户分析** — 数据增强 + 评分匹配
3. **开发信** — WhatsApp/Email发送
4. **技术调研** — 产品选型
5. **报价** — 报价单生成
6. **跟进** — CRM记录

## 状态机
```
发现 → 分析 → 触达 → 跟进 → 报价 → 成交
  ↑                                    │
  └──────────── 循环 ←─────────────────┘
```

## 编排逻辑
- 每个角色独立执行
- 状态传递通过共享数据
- 失败重试机制
- 人工确认节点
