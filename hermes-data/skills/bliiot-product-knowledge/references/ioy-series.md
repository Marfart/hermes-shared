# IOy 远程IO模块 (BL190~BL193)

## 概述
IOy系列是BLIIOT的多功能可编程远程IO模块，支持DI/DO/AI/AO/RTD/TC信号，通过RS485或以太网连接。

## 型号
- BL190: Modbus TCP, 以太网
- BL191: OPC UA, 以太网
- BL192: Modbus TCP, RS485
- BL192Pro: Modbus TCP, RS485 (增强版)
- BL193: Modbus TCP, 以太网 (高端)
- BA190: BACnet/IP, 以太网
- BA190Pro: BACnet/IP, 以太网 (增强版)

## 命名规则
BL193-Y11-Y31 = BL193 host + Y11 as Y1 board + Y31 as Y2 board

## IOy vs ARMxy Y板价格差异
同一Y板型号在IOy文件和ARMxy文件中价格可能不同。BL193用IOy文件价格，BL310~BL460用ARMxy文件价格。
