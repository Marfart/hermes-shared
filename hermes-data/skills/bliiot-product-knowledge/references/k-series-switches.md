# K系列 / 交换机 / 隔离器 / IPM100 / BLIoTLink

## K系列 4G DTU
- K5/K5S: 4G DTU, RS485/RS232
- K6/K6S: 4G DTU, 增强版
- K9: 高端4G DTU
- 无线数据传输终端，支持Modbus MQTT

## BL16x 工业以太网交换机
- 5口/8口/16口
- 百兆/千兆
- PoE可选
- 网管型/非网管型

## BL15x 信号隔离器
- 模拟量信号隔离
- 4-20mA, 0-10V
- BL150/BL151/BL152/BL153/BL154/BL155

## IPM100 IP67 远程IO
- IP67防水防震
- 8路干接点输入
- RS485 (Modbus RTU)
- 工作温度: -20~+70°C

## BLIoTLink 协议转换软件
- BLIIOT自研工业协议转换软件
- 预装于所有ARMxy和ARM高性能网关
- 下游: 各品牌PLC协议, Modbus RTU/TCP Master, DL/T645
- 上游: Modbus TCP, MQTT, OPC UA, BACnet/IP, IoT云平台
- 免费: 1设备 + 256数据点
- 安装: chmod 777 BLIoTLink.bin && ./BLIoTLink.bin
- 相关软件: BLRAT(远程访问), QuickConfig(快速配置), Node-RED(BL118预装)
