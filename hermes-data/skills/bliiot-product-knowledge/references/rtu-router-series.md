# RTU/路由器系列 (S27x/R40/Mxxx)

## S275 4G RTU
- **处理器**: ARM Cortex-A7
- **4G**: 4G LTE (GPS可选)
- **IO**: 8DI + 6AI + 4Relay
- **RS485**: 1路
- **SMS**: 内置短信报警
- **温宽**: -20~+70°C
- **防护**: IP30
- **应用**: 电力监控、环境监测

## S271 4G RTU
- IO: 4DI + 4AI + 4Relay
- 温宽: -20~+70°C

## S270 4G RTU
- IO: 2DI + 2AI + 2Relay
- 温宽: -20~+70°C

## R40 工业4G路由器
- **处理器**: MIPS CPU 580MHz
- **4G**: 4G LTE (多频段: E/CE/AU/A/J)
- **WiFi**: 802.11 a/b/g/n
- **以太网**: 1WAN + 3LAN (10/100Mbps)
- **防火墙**: DMZ, DoS防护, IP/MAC/域名过滤, 端口映射, 存取控制
- **VPN**: IPsec/OpenVPN/L2TP
- **PoE**: 可选 (R40A/R40B)
- **IO**: R40A/R40B 有 1xRS485 + 1xRS232 + 2DI + 2DO
- **防护**: IP30
- **特点**: 路由器+防火墙+IO 三合一

## Mxxx 以太网IO模块
- M120E, M130E, M140E, M150E, M160E, M200E~M240E
- 内置DI/DO/AI/AO (多种组合)
- 以太网连接，可独立工作
- M160E: Sample=$157

## S130/S150 GSM SMS控制器
- S130: 2DI+2Relay, SMS报警
- S150: 8DI+2Relay, SMS报警
- 区域版本: E, CE, AU, A, J
