## 不可跳过的产品线

以下产品线最容易漏掉但对客户需求至关重要：

1. **IOy系列** — 边缘I/O模块，支持DI/AI/DO/AO/RTD/TC，RS485或以太网连接
2. **MxxxT系列** — 以太网IO模块，自带8DI+8DO+4AI+2AO等多种型号组合
3. **R40路由器** — 内建防火墙(DMZ/DoS防护/IP过滤/MAC过滤/端口映射/存取控制) + VPN(IPsec/OpenVPN/L2TP)
4. **R40路由器 RS485/DI/DO/AI版** — 路由器+IO+协议转换三合一产品
5. **BL118** — Node-RED预装 + X/Y板扩展I/O

## 扫描方法

```bash
# 第一步：列出所有产品线
find "产品规格书/英文资料" -type d -maxdepth 2 | sort

# 第二步：从每个产品线找规格文档
ls "产品规格书/英文资料/<产品线>/<子目录>/"

# 第三步：对每个DOCX提取处理器/协议/I/O关键信息
```

## 如果客户需要还没找到

必须说"已扫全部产品线，未找到"。不要假设某个功能不存在。
