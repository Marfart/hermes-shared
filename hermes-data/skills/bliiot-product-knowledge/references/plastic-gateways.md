# 塑胶壳网关 (BL10x/BE10x/BA10x/BL120/BL121)

## BL10x 系列 (BA/BE/BL)
- **处理器**: 300MHz ARM MCU
- **RAM**: 128MB, **Flash**: 64MB
- **以太网**: 2xRJ45 10/100Mbps
- **串口**: 2CH RS485/RS232
- **4G**: 可选
- **供电**: DC 9~36V
- **尺寸**: 30x83x110mm
- **保护**: ESD +/-6kV, EFT 1kV

### BL10x 子系列
- **BL10x**: Modbus RTU/TCP -> MQTT/OPC UA (通用)
- **BE10x**: 能源/电力 IEC104 协议
- **BA10x**: 楼宇 HVAC (BACnet)
- **BL103**: OPC UA/BACnet Gateway

## BL120 — Modbus 网关
- 型号: BL120, BL120AC, BL120BN, BL120DT, BL120ML, BL120PM (PoE版)
- 协议: Modbus RTU/TCP <-> MQTT/OPC UA

## BL121 — OPC UA 网关
- 型号: BL121, BL121AC, BL121BN, BL121DT, BL121ML, BL121P, BL121PO, BL121POP
- WiFi: 可选 802.11a/b/g/n, 2.4G+5G
- 协议: Modbus RTU/TCP <-> OPC UA + MQTT

## BL122/BL123/BL124 — 专用协议网关
- BL122: Modbus <-> PROFINET
- BL123: Modbus <-> EtherCAT
- BL124: Modbus <-> Ethernet/IP

## BE120 — IEC 61850 网关
- **处理器**: 双核Cortex-A7 @1.2GHz
- **RAM**: 1GB, **存储**: 8GB
- **串口**: 4~8路隔离RS485/232
- **协议**: Modbus RTU/TCP <-> IEC 61850 (SCL文件配置)
- **温宽**: -45°C ~ 80°C
- **特点**: 双网口+可选WiFi/4G/5G, 钛合金外壳
