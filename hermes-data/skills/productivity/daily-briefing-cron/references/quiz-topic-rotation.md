# Investment Quiz Topic Rotation

The daily quiz in Kali's briefing rotates through these practical investing topics. Never use general finance trivia.

## Daily rotation

| Day | Topic | Example question |
|-----|-------|-----------------|
| Monday | **Fund basics** | What's the difference between an index fund and an actively managed fund? How are expense ratios calculated? What's an ETF? |
| Tuesday | **Investment strategy** | What is dollar-cost averaging? How does grid trading work? Value vs growth investing? When to use each? |
| Wednesday | **Risk management** | How does position sizing work? What's a proper stop-loss strategy? Why diversify across uncorrelated assets? The Kelly criterion. |
| Thursday | **Valuation metrics** | How to read PE, PB, ROE, dividend yield? What's a reasonable PE for different sectors? PEG ratio. EV/EBITDA. |
| Friday | **Asset allocation** | The 60/40 portfolio. How to rebalance? When to shift between stocks and bonds? Age-based allocation. The endowment model. |

## Question format rules

1. **Must be a practical skill** the user can apply, not theoretical knowledge
2. **4 multiple-choice options** (A/B/C/D), single correct answer
3. **Detailed explanation** — why each wrong option is wrong, why the right one is right
4. **Real-world application** — how this concept affects actual investment decisions
5. **Investing quote of the day** — one memorable principle to close

## Multi-question sessions

The user explicitly asked for more quiz content in a single delivery ("再加一些财经题目一起"). When the briefing is delivered as a PDF, include **5-6 questions** spanning different topics from different days. For example:

- Q1: Portfolio rebalancing (Friday topic)
- Q2: Dollar-cost averaging (Tuesday topic)
- Q3: PE ratio interpretation (Thursday topic)
- Q4: Stop-loss strategy (Wednesday topic)
- Q5: Behavioral finance / FOMO (Saturday/Sunday topic)
- Q6: Asset allocation by age (Friday topic)

This gives the user a full "lesson set" for the week in one PDF section.

## Anti-patterns

❌ "What company had the highest market cap in 2024?" — trivia, not practical
❌ "What is GDP?" — too general, not investment-specific
❌ Questions about specific stock tickers — datesit too fast
✅ "When building a portfolio, what's the main advantage of asset allocation?" — practical skill
✅ "If a stock has a PE of 50 in a sector averaging 20, what does that suggest?" — practical analysis
