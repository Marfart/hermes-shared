## 📝 一、雅思写作（IELTS Writing）

### 1.1 Task 1 小作文模板框架

| 题型 | 核心句型 | 关键表达 | 字数 |
|------|---------|---------|------|
| 折线图趋势 | rose/dropped/plateaued at... | peaked at X in Year | 150-180 |
| 柱状图对比 | A was X, while B was Y | significantly higher/lower | 150-180 |
| 饼图占比 | accounted for X% | the largest proportion | 150-180 |
| 表格数据 | A had X compared to B's Y | by contrast, likewise | 150-180 |
| 流程图 | the process begins with...then... | first step involves | 150-180 |
| 地图变化 | X was replaced by Y | A was converted into B | 150-180 |

### 1.2 Task 2 大作文框架（4种题型）

#### 🔵 观点型（Agree/Disagree）
```
1. 引言：Paraphrase + 明确立场
2. 主体1：支持方论证 × 逻辑链
3. 主体2：反方让步 + 反驳
4. 结论：重申立场
```

#### 🟢 利弊型（Advantages/Disadvantages）
```
1. 引言：背景介绍 + 利弊总览
2. 主体1：利处论证（2-3点）
3. 主体2：弊处论证 + 是否outweigh
4. 结论：综合判断
```

#### 🟡 问题解决型（Problems/Causes + Solutions）
```
1. 引言：问题背景
2. 主体1：问题表现/原因分析（2-3点）
3. 主体2：解决方案（每个原因对应1-2个方案）
4. 结论：需要多方努力
```

#### 🔴 双边讨论型（Discuss Both Views）
```
1. 引言：两种观点概述
2. 主体1：观点A论证（承认合理之处）
3. 主体2：观点B论证 + 自己倾向
4. 结论：平衡总结 + 个人立场
```

### 1.3 高分表达库

| 类别 | 普通表达 | 高分替代 |
|------|---------|---------|
| 原因 | because | due to/owing to/on account of/as a consequence of |
| 结果 | so | therefore/consequently/thus/as a result/accordingly |
| 转折 | but | however/nevertheless/nonetheless/whereas/whilst |
| 举例 | for example | for instance/to illustrate/a case in point is |
| 强调 | important | crucial/vital/essential/paramount/indispensable |
| 数量 | many | numerous/a myriad of/countless/substantial |
| 观点 | I think | I contend/I firmly believe/It is my conviction that |

## 🗣️ 二、雅思口语（IELTS Speaking）

### 2.1 Part 1 个人话题（20+常见话题）

| 话题类别 | 常考题目 | 答题策略 |
|---------|---------|---------|
| 🏠 居住 | Where do you live? | 2-3句：地点+特点+感受 |
| 💼 工作/学习 | What do you do? | job/study内容+选择原因 |
| 🎵 爱好 | Do you like music? | 喜欢什么+什么时候听+为什么 |
| 🌍 旅行 | Do you like traveling? | 去过哪里+下个想去的地方 |
| 📱 科技 | Do you use social media? | 频率+用途+评价 |
| 🍳 饮食 | What's your favorite food? | 是什么+为什么+何时吃 |

**Part 1 黄金策略：Direct Answer + Reason + Example (DRE)**
- Direct Answer: 直接回答Yes/No + 核心观点
- Reason: 1-2句解释原因
- Example: 具体例子或个人经历

### 2.2 Part 2 话题卡模板（Cue Card）

**万能结构（1分钟准备→说满2分钟）：**
```
1. 开头句（5秒）: "I'd like to talk about..."
2. What/Who（15秒）: 描述对象
3. When/Where（15秒）: 时间地点背景
4. Why（30秒）: 为什么印象深刻/重要
5. Feelings（30秒）: 你的感受/收获
6. 结尾升华（15秒）: 如果有机会会怎样
```

**5大高频话题类别及万能素材：**
| 类别 | 常考题目 | 核心素材 |
|------|---------|---------|
| 👤 人物 | Describe a person you admire | 家人/老师/朋友 → 品质+故事 |
| 🏞️ 地点 | Describe a place you visited | 图书馆/公园/咖啡馆/海边 |
| 📦 物品 | Describe something you bought | 手机/书/礼物/纪念品 |
| 🎬 经历 | Describe an important event | 考试/比赛/旅行/聚会 |
| 🎯 目标 | Describe a goal you achieved | 学英语/减肥/考雅思 |

### 2.3 Part 3 抽象讨论（双向辩论）

| 题目类型 | 分析框架 | 常用表达 |
|---------|---------|---------|
| 对比今昔 | Technology → changed lifestyle | In the past...but nowadays... |
| 利弊分析 | For individuals vs For society | On one hand...on the other hand |
| 不同年龄段 | Young people vs Elderly | While young people tend to... |
| 城乡差异 | Urban vs Rural areas | There's a stark contrast between... |
| 未来预测 | Based on current trends | It's conceivable that...will... |

## 👂 三、雅思听力（IELTS Listening）

### 3.1 四大Section分析

| Section | 话题范围 | 题型 | 难点 | 满分策略 |
|---------|---------|------|-----|---------|
| S1 社交对话 | 预订/报名/咨询/购物 | 填空（个人信息） | 拼写+数字 | 预读扫空格类型（日期/人名/电话/地址） |
| S2 独白 | 景点介绍/活动安排/导览 | 选择+配对+地图 | 方位描述+干扰项 | 圈出定位词+预判方向词（opposite/behind/north） |
| S3 学术讨论 | 课题讨论/作业设计/案例分析 | 选择+配对+填空 | 多人对话+观点切换 | 标记每个说话人+记录观点变化 |
| S4 学术讲座 | 自然科学/人文社科/科技 | 填空（笔记摘要） | 长难句+信息密度大 | 预读全文框架+抓信号词（firstly/another key point） |

### 3.2 精听训练法（Shadowing + Dictation）

**四步精听法（每天30分钟）：**
```
Step 1: 泛听（5min）- 正常速度听全文，理解大意
Step 2: 听写（15min）- 逐句暂停，写下听到的内容
Step 3: 纠错（5min）- 对照原文，标记听错/没听到的地方
Step 4: 影子跟读（5min）- 跟读录音，模仿语音语调
```

## 📖 四、雅思阅读（IELTS Reading）

### 4.1 12种题型攻略

| 题型 | 解题策略 | 耗时建议 |
|------|---------|---------|
| ✅ 判断题 (T/F/NG) | 定位→对比→判断(找到明确否=NO/无信息=NG) | 1.5min/题 |
| 📝 填空题 | 定位→判断词性→匹配同义替换 | 1min/题 |
| 🔗 匹配题(段落信息) | 扫读段落主题→题干关键→回找 | 2min/题 |
| 📋 标题匹配 | 首末句+转折词→总结段落大意 | 2min/题 |
| ✅ 选择题 | 排除法→回文定位确认 | 1.5min/题 |
| 🔄 摘要填空 | 上下文+词库预判词性 | 1.5min/题 |
| 📊 图表标注 | 定位原文描述→精确对应 | 1min/题 |
| 🔤 简答题 | 定位+注意字数限制 | 1.5min/题 |

### 4.2 平行阅读法（最优时间分配）

```
60分钟三篇文章分配：
┌─────────────────────────────────────┐
│  Passage 1 (简单·13题)  →  15分钟   │
│  Passage 2 (中等·13题)  →  20分钟   │
│  Passage 3 (最难·14题)  →  20分钟   │
│  检查涂卡                     5分钟   │
└─────────────────────────────────────┘
```

**平行阅读法步骤：**
1. 看1-2题题干，圈定位词（专有名词/数字/生僻词）
2. 扫读文章找定位词 → 找到后精读该句/段
3. 答题 → 立刻看下一题定位词
4. 一个段落可能含多个答案 → 一段读完再看下一段

### 4.3 同义替换速记本

| 原词 | 阅读常见替换 |
|------|------------|
| important | crucial/vital/significant/paramount/essential |
| change | transform/shift/alter/modify/convert |
| increase | rise/grow/soar/escalate/surge |
| decrease | decline/drop/fall/plummet/plunge |
| cause | lead to/result in/give rise to/trigger/spark |
| problem | issue/difficulty/challenge/hindrance/obstacle |
| solve | tackle/address/resolve/remedy/counter |
| different | diverse/varied/distinct/disparate/contrasting |

## 📋 七、90天备考计划

### Phase 1: 基础建设（Day 1-30）
- 每日：单词50新+50旧（Anki）
- 听力：S1/S2限时训练 + BBC 6min精听
- 阅读：P1限时训练 + 积累同义替换
- 写作：Task1 各题型模板背诵
- 口语：Part1 话题练习

### Phase 2: 强化提升（Day 31-60）
- 每日：单词30新+100旧（Anki）
- 听力：S3/S4精听 + 听力全真模拟
- 阅读：P1+P2限时 + 平行阅读法练习
- 写作：Task2 各题型框架 + 每周4篇限时写作
- 口语：Part2 话题卡练习 + 录音自评

### Phase 3: 冲刺模考（Day 61-90）
- 每日：单词复习150
- 隔日：整套模拟考试（限时3h）
- 写作：每日1篇+Grammarly/LanguageTool检查
- 口语：AI模拟全真考试（IELTS Speaking Simulator）
- 错题：各科错题本复习
