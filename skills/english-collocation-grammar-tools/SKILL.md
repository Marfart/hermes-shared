## 🎯 雅思语法+搭配要点

### 写作必备搭配
| 类型 | 示例 |
|------|------|
| 因果 | give rise to / contribute to / result in |
| 对比 | on the contrary / in contrast / whereas |
| 让步 | admittedly / it is true that / despite |
| 举例 | a case in point / for instance / namely |
| 总结 | in conclusion / to summarize / overall |

### 高分语法结构
1. **倒装句**：Not only does this...but it also...
2. **虚拟语气**：Were governments to invest...
3. **分词结构**：Having considered..., I believe...
4. **强调句**：It is...that...
5. **定语从句**：..., which means that...

## 🛠️ 使用建议

1. 用 **fraze-finder** 分析自己写的作文，找出搭配是否地道
2. 用 **LanguageTool** 检查语法错误
3. 用 **Harper** 做本地离线语法检查（不依赖网络）
4. 背诵 **The-English-Os** 的600搭配+600短语动词
5. 阅读时用 **fraze-finder** 高亮搭配，培养语感
